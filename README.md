#  Exercícios Realizados
## 1️ O Fechamento do Caixa

Simula de forma básica algo proximo de um caixa de um mercado qualquer.
O programa:
 lê o **nome de um produto**
 lê o **preço unitário**
 lê a **quantidade comprada**
Depois calcula o valor total da compra e exibe uma mensagem informando quanto custou o a compra no total.

---

## 2️ Conversor de Temperatura (Fahrenheit)
Neste exercício foi criado um conversor de temperatura.

O programa:
* recebe uma temperatura em **graus Celsius**
* converte para **Fahrenheit**
Utilizando a fórmula:

F = (C × 1.8) + 32

Esse tipo de conversão é bem usada em sistemas meteorológicos utilizados em países que utilizam Fahrenheit como os USA.

---

## 3️ Média Ponderada

Simula o cálculo de notas apartir de pesos nas notas.
O programa:
 pega duas notas
 aplica pesos diferentes para cada prova
Pesos usados:

 **Prova A - peso 4**
 **Prova B - peso 6**

A média é calculada utilizando a fórmula de média ponderada.

Média P = ( x peso * x nota) + (y peso * y nota)... / ( x peso + y peso)

---

## 4️ Consumo do Carro

Este exercício calcula o consumo médio de combustível de um veículo

O programa:
 lê a **distância percorrida em quilômetros**
 lê a **quantidade de combustível gasto em litros**
Depois calcula o consumo médio utilizando:
consumo = distância / combustível
O resultado é exibido em **km/l**.

---

## 5️ Cálculador de Desconto simples

Simula um sistema de descontos

O programa:
 lê o **preço original do produto**
 lê o **percentual de desconto**

Depois calcula:
 o **valor do desconto em reais**
 o **preço final com desconto aplicado**

Esse tipo de cálculo é comum em sistemas de vendas online.

#  Documentação – Importância do Tipo de Dados

Ao desenvolver algoritmos, e sempre bom escolher o tipo de dado utilizado para armazenar informações.

Existem dois tipos muito utilizados:

**Inteiro (int):**

 utilizado para números sem casas decimais
 exemplo: idade, quantidade de produtos, número de algo

**Real :**

 utilizado para números com casas decimais
 exemplo: preços, temperaturas, médias 

Escolher o tipo correto evita problemas de **arredondamento ou perda de precisão**, principalmente em sistemas financeiros ou científicos, onde pequenos erros podem causar resultados incorretos.

Por exemplo:

 valores monetários precisam manter precisão nas casas decimais
 cálculos científicos exigem grande precisão numérica

Por isso, é essencial utilizar **variáveis do tipo real para cálculos que envolvem números decimais**.
